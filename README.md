## 准备

在使用本插件前请确保 Docker CLI 以及 Docker Compose Plugin 已经安装，且可以从命令行直接使用。

详细安装方法请参考 [Docker 文档](https://docs.docker.com/engine/install/)

官方 Linux 快速安装一键脚本：

```bash
curl -fsSL https://get.docker.com | sudo sh
```

## 安装插件

```bash
nb self install nb-cli-plugin-docker
```

## 使用插件

```bash
nb docker
# 其他别名
# nb deploy
# nb compose
```
